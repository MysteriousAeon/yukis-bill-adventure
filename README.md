# Yuki's Bill Adventure

A Frogger-style arcade game created in Godot 4. Help Yuki the Bunny dodge gas, water, and electricity bills to reach financial freedom!

<img width="1280" height="720" alt="YukisBillAdventure_vCbDQXsQWU" src="https://github.com/user-attachments/assets/6edb3400-19a1-4849-ac10-d7a459711ef8" />

## How to Play
* **Arrow Keys / WASD:** Hop around.
* **Goal:** Reach the finish line without getting hit by the flying bills.
* **Esc:** Quit the game.

## Features
* Custom Grid-based Movement System
* High Score Save/Load System
* Randomized Enemy Spawning
